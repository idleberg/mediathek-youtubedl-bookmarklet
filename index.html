🍿 MediathekViewWeb Bookmarklet

This bookmarklet generates a list of youtube-dl commands for search results on MediathekViewWeb. To install it, drag the link below into your browser's bookmark bar.

mediathek-dl

Once installed, search for any movie in the Mediathek, then click the bookmarklet. This will copy all download commands to your clipboard (Chrome) or print it in the alert message (Firefox).
